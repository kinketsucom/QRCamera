# Name
QRカメラ

# Overview
<a href="https://github.com/kinketsucom/QRCamera/blob/master/QRCameraApp/QRCameraApp/bin/Release/QRCameraApp.exe">
QRカメラ</a>  
リンク先のView Rawリンクを押してファイルを保存->実行
してください。QRカメラが起動します。

## Description
QRコードをかざして、読み取りボタンを押すことでQRコードを読み取ります。
読み取るQRコードはローカルのディレクトリ(C:/hoge/piyo)を想定しており、
ディレクトリが存在する場合はそのディレクトリを開き、
存在しない場合はディレクトリを作成して開きます。
